Подробная установка данного шаблона 

1) Первым шагом устанавливаем терминал PowerShell , например отсюда https://www.comss.ru/page.php?id=6214

ПУНКТЫ 2,3 И 5 ВЫПОЛНЯЮТСЯ ЕСЛИ ВЫ УСТАНАВЛИВАЕТЕ GULP В ПЕРВЫЙ РАЗ

2) После скачивания, запускаем его от права администратора и вводим команду Set-ExecutionPolicy RemoteSigned 
Данный терминал чаще всего не даёт комментарии по поводу выполнения, поэтому если не вылезло никаких ошибок 
и курсор перешёл на следующую строку и готов принимать новую команду - значит всё прошло успешно.

3) Далее нам нужно установить gulp глобально,чтобы всё отрабатывало. Для этого мы запускаем npm install --global gulp-cli

4) Далее мы создаём папку с проектом. После этого нам нужно запустить этот терминал прямо в папке проекта.
Самый простой способ сделать это - зайти опять от права администратора и ввести следующую команду:
Set-Location -Path и указываем путь к папке.

5) После этого важно сразу выполнить npm install --global windows-build-tools , чтобы запуск прошёл удачно.
Ждём пока курсор будет готов принимать новую команду и процесс установки завершится
(загрузка этих инструментов может занять долгое время - не прерывайте процесс).

6) Затем выполняем команду npm i и ждём пока все пакеты будут установлены. Здесь может возникнуть несколько ошибок.
Если ошибка связана с gulp sass то выполняем npm rebuild node-sass и снова запускаем npm i. Если выдаёт ошибки,
связанные с Python, то просто игнорируем их.

7)После этого необходимо выполнить доп команду npm install webp-converter@2.2.3 --save-dev

8)Далее выполняем комманду gulp.В текущем шаблоне были ошибки,связанные с пакетом gulp imagemin который 
минифицирует картинки и переводит изображения в браузере автоматически в формат webp, если это возможно.
В результате поиска ошибки, выяснилось, что она скорее всего связана с тем, что происходит конфликт 
с node 6.3.1 и более старшими версиями.Переменные для данного плагина и pipe в функции images не удалены,
а закомментированы. Если есть желание проверить самому, вы можете раскомментировать их и попробовать выполнить 
комманду gulp - может быть что-то получится.


Если вы проделали всё точно по пунктам, у вас должна открыться страница браузера.Успехов!
